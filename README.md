# Spam Classifier Streamlit App 📧

This is a **Spam/Ham email and SMS classifier** built with **Python, NLTK, Scikit-learn**, and **Streamlit**.  
It uses a **TF-IDF vectorizer** and **Naive Bayes model** to predict whether a message is spam.

---

## Features

- ✅ Classifies emails and SMS as **Spam** or **Ham**
- ✅ Preprocesses text with **tokenization, stopwords removal, and stemming**
- ✅ User-friendly **Streamlit web interface**
- ✅ Can show **spam probability** (optional upgrade)
